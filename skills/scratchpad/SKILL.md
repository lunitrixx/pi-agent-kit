# Scratchpad

A persistent checklist that survives sessions. Stored in `.pi/memory/scratch.md`.

## Commands

- `/memory scratch` — show all items
- `/memory scratch add <item>` — add a new TODO
- `/memory scratch global` — show global scratchpad (cross-project)

## When to use

- User says "remind me to..." or "don't forget..."
- You discover a follow-up task during work
- Before ending a session — list open items for next time
- After compaction — check what was in progress
